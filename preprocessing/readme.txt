# トリガー検出閾値の仕様変更について (Trigger Threshold Specifications)

本ディレクトリには、実験環境ごとのオーディオ出力レベルの変動に対応するため、異なる検出感度を持つ2種類のスクリプトを格納している。

## 使用ガイドライン

### 1. Standard版（閾値: -1000）
- **基本運用:** 原則として本バージョンを使用する。
- **特徴:** ノイズ耐性が高く、環境ノイズによる誤検知（False Positive）のリスクが低い。
- **適用条件:** オシロスコープ等で信号を確認し、トリガー振幅が十分に確保されていることを推奨する。

### 2. low_threshold版（閾値: -400）
- **適用ケース:** ハードウェアの制約や音量設定の問題により、トリガー信号の振幅が減衰している場合に使用する。
- **注意点:** 閾値をゼロ近傍（-400）まで緩和しているため、感度は向上しているが、電気的ノイズや環境音を誤ってトリガーとして検出する可能性がある。使用時はデータ収集後の波形確認を慎重に行うこと。
